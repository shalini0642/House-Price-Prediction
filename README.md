# 🏠 House Price Prediction

A Machine Learning web application that predicts house prices using the California Housing Dataset.

## Features

- Predicts house prices based on housing features
- Interactive web interface built with Streamlit
- Machine Learning model trained using Linear Regression
- Simple and beginner-friendly implementation

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Dataset

California Housing Dataset provided by Scikit-learn.

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/House-Price-Prediction.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Train the model

```bash
python train.py
```

Run the application

```bash
streamlit run app.py
```
